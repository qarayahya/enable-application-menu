Adds an application menu dropdown to the WordPress block editor with quick access to dashboard, styles, templates, patterns, media, and plugins.

== Description ==

Enable Application Menu adds a convenient dropdown menu to the WordPress block editor toolbar. This menu provides quick access to commonly used admin pages without leaving the editor interface.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/enable-application-menu` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open any post or page in the block editor to see the new application menu button in the toolbar.

== Frequently Asked Questions ==

= Where does the menu appear? =

The application menu appears in the top-left corner of the block editor toolbar, marked with the WordPress logo icon.

= Can I customize the menu items? =

The current version includes a fixed set of commonly used admin pages. Future versions may include customization options.

= Does it work with custom post types? =

Yes! The menu is available in the block editor for all post types.

= Is it compatible with the Site Editor? =

No, the menu works only on the post editor.
